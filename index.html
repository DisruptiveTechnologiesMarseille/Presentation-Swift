## Swift ? - Langage créé par Apple pour iOS et OSX - Pour remplacer Objective C
### Objective C ``` NSString *str = @"hello,"; str = [str stringByAppendingString:@" world"]; ``` ### Swift ``` var str = "hello," str += " world" ```
